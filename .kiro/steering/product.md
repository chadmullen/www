# Product Overview

Personal blog for Chad Mullen, an Senior AWS Solutions Architect. The site focuses on cloud architecture, AI/ML, and infrastructure modernization topics.

## Purpose

- Share technical knowledge and learning experiences
- Demonstrate AWS architecture patterns
- Maintain a professional online presence

## Key Features

- Static blog with Hugo-generated content
- Fast, CDN-cached delivery via CloudFront
- Privacy-friendly analytics (Cloudflare Web Analytics)
- Search functionality
- Responsive design with PaperMod theme
- Cost-optimized AWS hosting (< $1/month)

## Target Audience

Technical professionals interested in AWS, cloud architecture, and infrastructure topics.
